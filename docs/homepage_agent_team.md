# WOOKVAN 홈페이지 제작 에이전트 팀 구성

## 개요

WOOKVAN 에이전트 팀이 고객 홈페이지를 제작하는 시스템.
대표님이 고객 소통 + 최종 납품을 담당하고, 에이전트 팀은 내부 제작을 수행.

---

## 도구 사용 규칙 (필수)

**에이전트가 직접 MCP/Playwright/WebSearch 등 고비용 도구를 호출하지 않는다.**
토큰 절약을 위해 모든 외부 작업은 n8n WF를 통해 수행한다.

### 원칙
1. **n8n WF = 에이전트의 도구** — 에이전트는 n8n webhook 호출로 작업 수행
2. **직접 도구 호출 금지** — Playwright, WebSearch, WebFetch, MCP 직접 사용 금지
3. **코드 작성은 직접** — Read/Edit/Write/Bash는 에이전트가 직접 수행 (토큰 효율적)
4. **새 도구 필요 시** → WF 제안서 제출 → 대표님 승인 → WF 생성

### WF 제안서 양식
```
제안 에이전트: (PM/리서치/디자인 등)
WF 이름: (예: 참고사이트 분석기)
용도: (무엇을 자동화하는지)
입력: (webhook으로 받을 데이터)
출력: (결과물 형식)
필요 노드: (Playwright, HTTP Request 등)
```

### 에이전트별 허용 도구

| 에이전트 | 직접 사용 OK | n8n WF 경유 |
|----------|------------|-------------|
| PM | TaskCreate/Update, Discord 보고 | Notion 견적DB, 스케줄링 |
| 리서치 | 결과 정리/문서화 | 웹 크롤링, 사이트 분석, 스크린샷 |
| 기획UX | 문서 작성, 사이트맵 | Notion 페이지 생성 |
| 디자인 | HTML/CSS/Tailwind 코드 작성 | 시안 스크린샷 검증 |
| 프론트엔드 | Next.js/React 코드 작성 | 빌드/배포, Lighthouse |
| 백엔드 | API/DB 코드 작성 | Notion CMS 연동, 외부 API |

## 팀 구성 (6명)

### ① PM 에이전트 (내부 총괄)
- **역할**: 프로젝트 일정 관리, 에이전트 작업 분배, 품질 검수, 대표님 보고
- **직접 도구**: TaskCreate/Update, Discord 보고 (discord_reply.py)
- **n8n WF**: Notion 견적DB CRUD, 가견적 자동산출
- **업무**:
  - 견적 폼 접수 시 가견적 자동 산출
  - Notion 견적 페이지 생성 → Discord로 대표님에게 링크 전송
  - 스프린트 계획 수립 + 진행상황 추적
  - 에이전트 간 작업 순서 조율
  - 통합 테스트 + 최종 검수 후 대표님 보고
- **고객 대면**: 없음 (내부 전용)

### ② 리서치 에이전트
- **역할**: 트렌드 분석, 참고사이트 조사, 구현 방법론 조사
- **직접 도구**: 분석 결과 문서화 (Read/Write)
- **n8n WF**: 웹 크롤링 WF, 사이트 분석 WF, 스크린샷 WF
- **업무**:
  - 최신 웹 디자인/기술/UX 트렌드 분석
  - 고객 업종별 경쟁사 사이트 분석 (레이아웃, 기능, 디자인 패턴)
  - 우리 에이전트 팀이 제작 가능한 범위 내 최적 구현 방법 조사
  - 참고사이트 분석 레포트 작성
  - **기획UX 에이전트(프롬프트 엔지니어)에게 구체적 방향/노하우 전달**
- **산출물**: 리서치 리포트 (트렌드 분석 + 참고사이트 분석 + 구현 가이드)
- **필요 WF (제안 대상)**:
  - 참고사이트 스크린샷/분석 WF
  - 웹 트렌드 크롤링 WF

### ③ 기획/UX 에이전트 (프롬프트 엔지니어)
- **역할**: 리서치 결과 기반 사이트 설계, IA, 와이어프레임
- **직접 도구**: 문서 작성 (Read/Write), 사이트맵 작성
- **n8n WF**: Notion 페이지 생성/수정
- **업무**:
  - 리서치 리포트 수신 → 사이트맵 설계
  - 페이지별 콘텐츠 배치 + 사용자 흐름(UX Flow) 설계
  - 와이어프레임 작성 (페이지별 레이아웃)
  - 메뉴 구조 + 네비게이션 설계
  - 각 에이전트에게 구체적 제작 프롬프트/지시 전달
- **산출물**: 사이트맵, 와이어프레임, UX 흐름도, 제작 가이드

### ④ 디자인 에이전트
- **역할**: UI 시안 제작, 비주얼 디자인
- **직접 도구**: HTML/CSS/Tailwind 코드 작성
- **n8n WF**: 시안 스크린샷 캡처 WF, 디자인 검증 WF
- **업무**:
  - 기획 가이드 기반 컬러팔레트 + 타이포그래피 결정
  - 메인페이지 시안 제작 (2~3종 → 대표님 선택)
  - 서브페이지 디자인 시스템 구축
  - 반응형 레이아웃 (모바일/태블릿/데스크톱)
  - UI 컴포넌트 라이브러리 (버튼, 카드, 폼 등)
- **산출물**: 디자인 시안, 컴포넌트 시스템, 스타일 가이드
- **필요 WF (제안 대상)**:
  - HTML → 스크린샷 변환 WF
  - 반응형 검증 WF (모바일/태블릿/데스크톱)

### ⑤ 프론트엔드 에이전트
- **역할**: 디자인 → 코드 변환, 페이지 구현
- **직접 도구**: Next.js/React/Tailwind 코드 작성, Bash (npm/빌드)
- **n8n WF**: 빌드 검증 WF, Lighthouse 성능 테스트 WF, 배포 WF
- **업무**:
  - 디자인 시안 코드 변환 (pixel-perfect)
  - 페이지별 구현 (메인 + 서브페이지)
  - SEO 최적화 (메타태그, OG태그, 구조화 데이터)
  - 반응형 구현 + 인터랙션/애니메이션
  - 크로스 브라우저 호환
  - 이미지 최적화 + 성능 튜닝
- **산출물**: 완성된 웹페이지 코드
- **필요 WF (제안 대상)**:
  - Lighthouse 자동 테스트 WF
  - Vercel 배포 트리거 WF

### ⑥ 백엔드/기능 에이전트
- **역할**: CMS 연동, API, 기능 개발
- **직접 도구**: Next.js API Routes 코드 작성, Bash (서버 테스트)
- **n8n WF**: Notion CMS 동기화 WF, 문의폼 처리 WF, 결제 연동 WF
- **업무**:
  - Notion CMS 연동 (콘텐츠 동적 관리)
  - 문의 폼 처리 (이메일/Discord 알림)
  - 회원관리 시스템 (필요시)
  - 결제 연동 (쇼핑몰형)
  - 갤러리/게시판 백엔드
  - 관리자 CMS 페이지
  - 배포 + 도메인/호스팅 설정
- **산출물**: API, CMS 연동, 기능 구현 코드
- **필요 WF (제안 대상)**:
  - Notion → 웹사이트 콘텐츠 동기화 WF
  - 문의폼 → 이메일/Discord 알림 WF

## 워크플로우

```
1. 고객 폼 제출
   ↓
2. PM: 자동 가견적 산출 → Notion 견적 페이지 생성 → Discord 링크 전송
   ↓
3. 대표님: 견적서 검토/수정 → 고객에게 전달
   ↓
4. 고객 승인 → 대표님이 PM에게 착수 지시
   ↓
5. 리서치: 트렌드 분석 + 참고사이트 조사 → 리포트 작성
   ↓
6. 기획UX: 리서치 기반 사이트맵 + 와이어프레임 → 대표님 확인
   ↓
7. 디자인: 시안 2~3종 제작 → 대표님 선택 → 고객 확인
   ↓
8. 프론트엔드 + 백엔드: 동시 개발 (2~3주)
   ↓
9. PM: 통합 테스트 + 최종 검수 → 대표님 보고
   ↓
10. 대표님: 최종 납품 → 고객 확인 → 유지보수 계약
```
